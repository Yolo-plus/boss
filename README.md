# 直聘前台

## 一、实现功能
用户注册/登录，求职者/BOSS列表，实时聊天，消息列表，个人详情等

## 二、技术应用
使用React全家桶 + ES6等技术

1. 下载依赖 - npm install
2. 启动前台 - npm run start（需要先启动后台https://github.com/Yolo-plus/boss_node）
3. localhost:3000
